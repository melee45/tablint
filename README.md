# csv-guard

Validate CSV files against a schema, fully offline, with local license enforcement.

## Usage

    csv-guard validate <csv_file> --schema <schema_file> [--json] [--quiet]

- `validate`: Primary command to validate a CSV file.
- `--schema`: Path to schema file (required for validation).
- `--json`: Output results in JSON format.
- `--quiet`: Suppress non-error output.

## Features
- Local CLI, no network calls
- License key enforcement (local, offline)
- Free and Pro tiers
- 50 validations/month for free tier
- Unlimited for Pro

## Limitations
- No APIs, telemetry, or payment processing
- No auto-fix or AI-based validation
